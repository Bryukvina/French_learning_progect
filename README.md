# Симулятор французского
Проект по реализации веб приложения на основе библиотеки Django

В данном приложении реализована возможность обучения с использованием карточек
Слова заносятся в базу, затем происходит обучение спереводрачиваем карточек. Также можно проходить игры на запомниание и проверку знаний.
